# SMS-Spam-Classification-with-MLflow
SMS Spam Classification using NLP (TF-IDF + Stemming) with MLflow for model comparison

preprocess->doing Text Cleaning saving x_train,y_train,x_test,y_test Label Encoding , Implementing TF-IDF
test1 -> doing Logistic Regression 
test2 -> naive bayes 
test3 -> svc
test4 -> xgb

every run log (Accuracy, Precision, Recall, and F1-Score)

mlflow ui --port 5000
